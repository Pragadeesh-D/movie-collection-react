# Movie Collection React App

A simple React web application that displays a list of movies using data fetched from the TMDb Collection API.

## Features
- Fetches real movie data from TMDb API
- Displays movies in a card-based layout
- Built using React functional components and hooks
- Responsive UI

## Technologies Used
- React
- JavaScript
- TMDb API
- HTML & CSS

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open http://localhost:3000 in the browser

## API Used
TMDb - Get Collection Details API
